# Installation

## Install

#### Development

	pipenv install sea-lib-phishlabs --dev
	
#### Production

	pipenv install sea-lib-phishlabs

## Upgrade

#### Development
	
	pipenv update --dev
	
#### Production

	pipenv update
	
## Firewall

* None

## Troubleshooting

* None

## Proxy

* Proxy needed as environment variable to access URI.

## Environment Variables

* `PL_USER` - Username
* `PL_PASS` - Password
* `PL_PROXY` - Phishlabs proxy
* `PL_URI` - API URI
