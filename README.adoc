== Build
.From Source Directory
[source,shell]
pipenv shell
python -OO setup.py sdist

This will create a .tar.gz file in /<source_root>/dist

Copy this to
[source,shell]
/opt/fitb/pip

and it will be available for install using the directions provided here.
